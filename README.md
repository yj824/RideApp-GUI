# RideApp GUI

A desktop ride-request application built in Java with a Swing-based graphical user interface.

## Features
- Rider, pickup, and dropoff input form
- Input validation for required fields and ZIP codes
- Ride request creation
- Ride queue display
- Driver assignment for unclaimed rides
- Dynamic fare calculation
- Express ride option
- Passenger range selection

## Technologies
- Java
- Swing
- Object-Oriented Programming (OOP)

## Project Structure
- `RideApp.java` - main GUI application and event handling
- `Ride.java` - ride data model and queue summary logic
- `Address.java` - address data model

## What I Learned
This project helped me practice Java GUI development, input validation, event-driven programming, and organizing application logic using object-oriented design.
